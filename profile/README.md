# Keviq

Keviq builds AI-native software products for modern work, developer productivity, and intelligent automation.

## What we are building
- **KeviqCoder** — AI coding tools for developers
- **KeviqCore** — the core platform for AI agents and workflows
- **KeviqERP** — AI-powered enterprise operations platform
- **KeviqTune** — lightweight model fine-tuning tools
- **KeviqBot** — robotics products for education and entertainment
- **KeviqFlow** — browser-based workflow automation tools

## Mission
To build practical AI systems that help people and organizations work faster, smarter, and with greater leverage.
